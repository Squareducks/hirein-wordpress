## Quick Start

### Step 1: Deploy the Template

```bash
# The template is already in the repository
# Commit and push to deploy to production
git add wp-content/themes/lakeside/page-compact-keyword.php
git commit -m "Add Compact Keyword page template for scaffold towers"
git push origin main
```

### Step 2: Create Pages in WordPress

**Option A: Manual Creation (Recommended for first 10)**

1. Log into WordPress admin
2. Go to Pages → Add New
3. Set Template to "Compact Keyword Landing Page"
4. Copy content from the pre-written pages
5. Set slug, meta description, and publish

**Option B: CSV Import (For bulk creation)**

1. Install "WP All Import" plugin
2. Upload `wordpress_pages_import.csv`
3. Map columns and run import

### Step 3: Configure SEO

For each page:
1. Set Yoast Focus Keyword
2. Verify Meta Description
3. Set Canonical URL
4. Enable breadcrumbs

### Step 4: Submit to Google

1. Open Google Search Console
2. Use URL Inspection tool
3. Request indexing for each new page

### Step 5: Track Rankings

Add keywords to rank tracking tool and monitor weekly.
