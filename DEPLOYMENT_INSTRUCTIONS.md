### Option B: CSV Import (For bulk creation)

1. Install "WP All Import" plugin in WordPress
2. Upload the file `wordpress_pages_import.csv`
3. Map the columns:
   - post_title → Page Title
   - post_content → Page Content
   - post_excerpt → Meta Description
   - post_name → Page Slug
   - template → Page Template
4. Run the import

## Step 3: Configure SEO

For each page, configure Yoast SEO:

1. Set Focus Keyword (from page title)
2. Verify Meta Description (should be auto-populated)
3. Set Canonical URL
4. Enable breadcrumbs

---

## Step 4: Submit to Google Search Console

1. Log into Google Search Console
2. Go to URL Inspection
3. Submit each new page URL for indexing
4. Request indexing
