# HireIn Custom Themes

## Active Theme: Lakeside

The HireIn website uses the Lakeside theme as its primary theme. This theme has been customized for the equipment rental business.

### Theme Structure

```
themes/
├── lakeside/              # Main Lakeside theme files
│   ├── style.css         # Main stylesheet
│   ├── index.php         # Main template file
│   ├── functions.php     # Theme functions
│   ├── header.php        # Header template
│   ├── footer.php        # Footer template
│   └── [other-files]     # Additional theme files
└── lakeside-child/       # Child theme (if exists)
    ├── style.css         # Child theme styles
    └── functions.php     # Child theme functions
```

### Customizations

The theme has been customized for:
- Equipment rental product displays
- WooCommerce integration
- SEO optimization with Rank Math
- Performance optimization with WP Rocket

### Installation

Theme files need to be extracted from the live site and added to this repository. The theme files are located in `/wp-content/themes/` on the live server.

### Note

Due to the file manager interface limitations, the actual theme files need to be extracted manually or through alternative methods. This README serves as a placeholder until the theme files can be properly extracted and committed to the repository.
