# Lakeside Theme

Custom WordPress theme for HireIn equipment hire business.

## Theme Information
- **Theme Name**: Lakeside Theme
- **Author**: Robert Nash
- **Version**: 1.0
- **Description**: Custom theme for Lakeside Hire equipment rental business

## Features
- WooCommerce integration for equipment hire
- Custom checkout processes
- Location-based functionality
- Equipment catalog management
- Responsive design with Bootstrap
- Custom styling for hire industry

## File Structure
```
lakeside/
├── style.css           # Main stylesheet with theme header
├── functions.php       # Theme functions and WooCommerce customizations
├── index.php          # Main template file
├── header.php         # Header template
├── footer.php         # Footer template
├── css/               # Additional stylesheets
├── js/                # JavaScript files
├── menu/              # Menu templates
├── woocommerce/       # WooCommerce template overrides
└── Feefo/             # Feefo integration files
```

## Dependencies
- WordPress 5.0+
- WooCommerce plugin
- Bootstrap CSS framework
- Custom fonts (Bree Serif)

## Customizations
- Multiple sidebar registrations
- Custom WooCommerce button text ("Hire Now" instead of "Add to Cart")
- Product downloads tab for attachments
- Custom image sizes for equipment photos
- Cart fragment updates for AJAX functionality
- Custom pricing display for variable products

## Installation
1. Upload theme files to `/wp-content/themes/lakeside/`
2. Activate theme in WordPress admin
3. Install required plugins (WooCommerce, etc.)
4. Configure theme settings and menus

## Notes
- This theme is specifically designed for equipment hire businesses
- Contains extensive WooCommerce customizations
- Includes custom post types and fields for equipment management
- Optimized for equipment rental workflow
